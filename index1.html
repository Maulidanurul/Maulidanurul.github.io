Dan sekarang saya berada di kelas XI-5 SMK-SMAK Bogor.
Kenapa saya memilih SMAKBo menjadi sekolah lanjutan saya setelah lulus dari SMP? Karena SMAKBo ini memiliki daya tarik untuk mengikat hati saya. Ya walaupun sekolah di SMAKBo itu menjadikan saya siswa yang kurang tidur dan memiliki kantung mata seperti panda. Selain faktor daya tarik, ada juga hal-hal yang menguatkan diri saya untuk menlanjutkan sekolah ke SMAKBo.

  • Dukungan orang tua
      Sewaktu saya bilang saya ingin melanjutkan ke SMAKBo, ayah dan ibu saya sangat senang mendengarnya. Jadi, ayah dan ibu sering menyemangati saya untuk masuk ke SMAKBo.
  • Keinginan sendiri
      SMAKBo itu adalah sekolah impian saya. Saya menyukai mata pelajaran kimia sejak saya SMP. Dan juga, dulu saya sering mendengar hal seperti SMAKBo sekolah yang sangat bagus, kalau lulus dari SMAKBo cepet dapat pekerjaan, dan masih banyak lagi.
  • Keluarga
      Saya itu anak pertama dan memiliki adik. Nah keinginan saya masuk SMAKBo juga saya ingin membantu orang tua saya untuk membiayai sekolah adik saya. Agar orang tua saya tidak berat dalam masalah biaya.
